# Password Audit Lab

A beginner cybersecurity project that demonstrates how weak passwords can be identified in a controlled offline environment using hashed credentials and a wordlist.

## Purpose
This project was created for education and authorized lab testing only. It does not target live systems, websites, or online accounts.

## Features
- SHA-256 hash comparison
- Dictionary-based password auditing
- Weak password detection
- Summary statistics
- Beginner-friendly Python code

## Files
- `audit.py` - main script
- `hashes.txt` - sample test hashes
- `wordlist.txt` - sample candidate passwords

## How to Run
```bash
python audit.py
